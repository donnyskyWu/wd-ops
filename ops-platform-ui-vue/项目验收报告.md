## 🚀 部署指南

### 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0 或 pnpm >= 8.0.0

### 开发环境

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问地址
http://localhost:3000
```

### 生产构建

```bash
# 构建生产版本
npm run build

# 分析打包体积
npm run build:analyze

# 本地预览
npm run preview
```

### Docker部署

```bash
# 构建镜像
docker build -t ops-platform-ui .

# 运行容器
docker run -d -p 80:80 ops-platform-ui
```
