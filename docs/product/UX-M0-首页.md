## 3. 页面 P-M0-001

### 3.1 布局区块

| 区块 | 组件 | 说明 |
|------|------|------|
| 顶栏 | IP 组筛选 + 日期范围 + 刷新 | 联动所有指标 |
| 快捷入口 | 横向卡片列表（最多 8 个） | 基于权限动态渲染 |
| 指标卡 | 4 列等宽卡片 | 总作者/内容数/SOP/绩效 |
| 图表区 | 2 列等宽 | 折线图 + 饼图 |
| 待办表格 | 标准表格 | 最多 10 条 |

### 3.3 状态矩阵

| 状态 | 条件 | 表现 |
|------|------|------|
| 空 | 待办 0 条 | "暂无待办" |
| 加载 | 首次进入 | 全屏 Skeleton（5 个接口并发） |
| 错误 | 任一接口失败 | 顶部 Banner + 重试 |
| 无权限 | 角色无首页权限 | 整页 "无访问权限" |
| 无 IP 组 | 用户无任何 IP 组权限 | 筛选器仅显示"全部"；数据按"全部"展示 |

### 3.4 关键交互

- **IP 组筛选联动**：切换 → 5 个接口重新调用 → loading → 全部刷新
- **平台筛选联动**：切换 → 仅图表接口重新调用
- **快捷入口点击**：跳转到对应页面（新窗口？同窗口？→ 同窗口 + 路由）
- **待办操作**：点击行 → 跳转到 action_url

---

## 4. 跨页通用约定

- IP 组选择器仅显示 `status=启用` 且用户有权限的 IP 组
- 日期范围选择：今天/近 7 天/近 30 天/自定义
- 快捷入口顺序：按用户角色 + 业务域优先级
- 指标卡数值超过 1w 用 "1.2w" 简写；超过 100w 用 "100w+"
