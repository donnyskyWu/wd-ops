## 2. 奥创通道 — 已确认 schema 摘要（ADR-045）

### 2.1 `oa_aocreate_api`（租户凭证，1:1 租户）

不变；**不**增加 `aochuang_account_id` 列。

### 2.2 `oa_aocreate_account`（新建 · 多账号子表）

| 列 | 说明 |
|----|------|
| `id` | PK |
| `tenant_id` | 租户隔离 |
| `aocreate_api_id` | FK → `oa_aocreate_api.id` |
| `account_name` | 运营可读名称 |
| `aochuang_account_id` | 奥创客服/管理端 accountId（path 参数） |
| `status` | ENABLED / DISABLED |
| `last_device_sync_at` | 最近设备同步 |
| `conn_status` | 连接诊断 |

**UK**：`(tenant_id, aochuang_account_id)`

### 2.3 `oa_personal_wechat_account` 扩展（设备绑定）

| 列 | 说明 |
|----|------|
| `aochuang_wechat_account_id` | 设备 ID；同步写入，默认只读 |
| `aochuang_account_ref_id` | FK → `oa_aocreate_account.id` |
| `aochuang_bind_status` | UNBOUND / AUTO / MANUAL / PENDING |
| `aochuang_nickname` / `aochuang_avatar` / `aochuang_is_alive` | 快照 |

**UK**：`(tenant_id, aochuang_wechat_account_id)` — 一设备一行（Q5）

### 2.4 奥创数据流

```
oa_aocreate_api
  → oa_aocreate_account[] (多 accountId)
  → GET /accounts/{accountId}/wechatAccounts
  → DeviceSyncService (匹配 / 待绑定)
  → oa_personal_wechat_account
  → friends → messages
```
