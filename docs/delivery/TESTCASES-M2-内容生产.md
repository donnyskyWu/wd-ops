## 2. SOP 模板（FR-M2-001）

### TC-M2-001-01 创建模板

- **步骤**：POST `/sop/template/create` `templateName="测试模板" contentType="SHORT_VIDEO" platformType="DOUYIN"`
- **预期**：200 OK + 返回 id

### TC-M2-001-02 DAG 验证 - 存在环

- **步骤**：节点 A `predecessors=[B]`，节点 B `predecessors=[A]`
- **预期**：错误码 2001 + 提示环

### TC-M2-001-03 DAG 验证 - 无环

- **步骤**：A→B→C
- **预期**：200 OK

### TC-M2-001-04 预置模板加载

- **前置**：全新租户
- **步骤**：执行 `db/init_sop.sql`
- **预期**：存在"标准内容生产运营流程"（14 节点）

### TC-M2-001-05 审核配置联动

- **步骤**：编辑节点 `need_review=1`，`reviewer_role` 留空
- **预期**：错误码 2003 + 提示 reviewer_role 必填

### TC-M2-001-06 字典选择器

- **步骤**：编辑节点 `executor_role`
- **预期**：弹出 `dict_position` 选择器

### TC-M2-001-07 删除引用中模板

- **步骤**：任务 A 引用模板 X，DELETE 模板 X
- **预期**：错误码 1502

### TC-M2-001-08 启用空模板

- **步骤**：启用无节点的模板
- **预期**：错误码 2005

### TC-M2-001-09 节点类型持久化（AC-M2-001-6，S-10）

- **步骤**：创建节点 `nodeType=CONTENT_GENERATION` → GET node/list
- **预期**：返回 `nodeType=CONTENT_GENERATION`；非法值 → 1503

## 3. SOP 任务（FR-M2-002）

### TC-M2-002-01 DAG 顺序激活

- **步骤**：节点 A `predecessors=[B,C]`，B 和 C 未完成
- **预期**：A 状态=PENDING，不可操作

### TC-M2-002-02 并行组同时激活

- **步骤**：节点 1/2/3 `parallel_group=GROUP_A`，前置节点完成
- **预期**：3 个节点同时显示 IN_PROGRESS

### TC-M2-002-03 SLA 超时

- **步骤**：节点 `sla_hours=1`，等待 2h
- **预期**：钉钉自动通知

### TC-M2-002-04 审核驳回回到执行中

- **步骤**：节点 A PENDING_REVIEW，审核人驳回
- **预期**：A 状态=IN_PROGRESS

### TC-M2-002-05 我的任务

- **步骤**：用张三账号登录，调用 `/task/my-tasks`
- **预期**：仅返回张三被分配的任务

### TC-M2-002-06 任务撤回

- **步骤**：任务 PENDING，调用 `撤回`
- **预期**：状态=CANCELLED

### TC-M2-002-07 执行人校验

- **步骤**：任务分配给张三，李四调用 `/task/{id}/start`
- **预期**：错误码 1501（无权操作）

### TC-M2-002-08 我的任务执行入口（AC-M2-002-5，✅ S-12）

- **步骤**：执行人登录，我的任务 PENDING → 点击「执行」
- **预期**：打开 `/task/{id}/execute`，展示基本信息

### TC-M2-002-09 内容生成完成门禁（AC-M2-002-6，✅ S-12）

- **步骤**：`nodeType=CONTENT_GENERATION`，无 COMPLETED 内容 → POST `execute/complete`
- **预期**：错误码 2008

### TC-M2-002-10 内容生成正常完成（AC-M2-002-7，✅ S-12）

- **步骤**：关联内容 `status=COMPLETED` → POST `execute/complete`
- **预期**：任务 → DONE 或 PENDING_REVIEW

## 4. 计划管理（FR-M2-009）

> **集成测试**：`M2PlanS09IT`（P0）

### TC-M2-009-01 创建草稿计划

- **步骤**：POST `/admin-api/oa/plan/create`，含 templateId、ipGroupId、competitions、全节点 steps
- **预期**：200 OK + 返回 planId；`get` 状态=DRAFT

### TC-M2-009-02 草稿任务列表不可见（AC-M2-009-1）

- **前置**：TC-M2-009-01 已创建
- **步骤**：GET `/admin-api/oa/task/list`
- **预期**：列表中无该 `planName` 的任务

### TC-M2-009-03 启动后任务可见（AC-M2-009-2）

- **步骤**：POST `/plan/{id}/start` → GET `/task/list`
- **预期**：该计划任务条数 = 步骤执行人总数；状态=PENDING

### TC-M2-009-04 终止审批全流程（AC-M2-009-3）

- **步骤**：POST `/terminate`（含 reason）→ 组长 POST `/terminate/approve`
- **预期**：计划状态=TERMINATED；关联任务=TERMINATED

### TC-M2-009-05 非草稿不可删除

- **前置**：已启动或已终止计划
- **步骤**：DELETE `/plan/delete?id=`
- **预期**：错误码 **2023**

### TC-M2-009-06 步骤分配赛事（AC-M2-009-4，✅ S-11）

- **步骤**：create 计划，step.competitionId=cmp-001 → start → GET task/list
- **预期**：关联任务 `competitionId=cmp-001`

## F+P+E 用例补充（v1.0 追加）

> 本节按 **AI 驱动产品开发方法论** § 5.2 要求，补充 内容生产 模块的 **功能（F）+ 权限（P）+ 异常（E）** 三类用例。

### TC-F 功能用例（3 条）

| ID | 标题 | 前置 | 预期 |
|----|------|------|------|
| TC-F-M2-001 | 内容创建-正常路径 | 必填字段完整+作者存在+平台字典合法 | 创建成功 |
| TC-F-M2-002 | SOP 任务-完成路径 | SOP 模板存在+任务状态为可完成 | 任务完成 |
| TC-F-M2-003 | 内容发布-正常路径 | 审核通过+发布平台字典合法 | 发布成功 |

### TC-P 权限用例（3 条）

| ID | 标题 | 前置 | 预期 |
|----|------|------|------|
| TC-P-M2-001 | 作者越权-创建他人内容 | A 创建 B 作者的内容 | 返回 403 错误码 1403 |
| TC-P-M2-002 | 审核员越权-发布未审核内容 | 发布状态=PENDING | 返回 403 错误码 1403 |
| TC-P-M2-003 | 跨租户-访问他租户内容 | tenantId 不一致 | 返回 1504 错误 |

### TC-E 异常用例（4 条）

| ID | 标题 | 前置 | 预期 |
|----|------|------|------|
| TC-E-M2-001 | 字典值非法-平台类型 | platform=dict_value_placeholder | 返回 1503 错误 |
| TC-E-M2-002 | 强关联缺失-作者不存在 | authorId=99999 | 返回 1501 错误 |
| TC-E-M2-003 | 强关联已绑定-作者被占用 | authorId 已被其他内容绑定 | 返回 1502 错误 |
| TC-E-M2-004 | 跨租户-作者非本租户 | author.tenantId 异 | 返回 1504 错误 |

### 错误码映射

- 1500：参数格式/必填校验
- 1501：强关联实体不存在
- 1502：强关联实体已绑定
- 1503：字典值不合法
- 1504：跨租户访问
- 1403：越权操作

详见 [`GLOBAL-CONVENTIONS.md § 4`](./GLOBAL-CONVENTIONS.md)
