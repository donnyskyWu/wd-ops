## 2. 功能测试

### 2.1 TC-M0-001 默认进入加载

- **步骤**：
  1. 用"运营组长"账号登录
  2. 进入首页
- **预期**：
  - 4 个指标卡 ≤ 1.5s 加载完成
  - 2 个图表渲染完成
  - 待办表格最多 10 条
- **来源**：AC-M0-001-1

### 2.2 TC-M0-002 IP 组筛选联动

- **步骤**：
  1. 进入首页，IP 组=全部
  2. 切换 IP 组为"八卦一组"
  3. 观察 4 指标 + 2 图表 + 待办
- **预期**：所有指标、图表、待办统一按"八卦一组"聚合
- **来源**：AC-M0-001-2

### 2.3 TC-M0-003 IP 组选择器仅显示启用 + 有权限

- **前置**：用"运营组长"账号
- **步骤**：
  1. 进入首页
  2. 点击 IP 组选择器
- **预期**：仅显示"八卦一组"（启用 + 有权限）；"娱乐八卦大组"不显示
- **来源**：AC-M0-001-3, GLOBAL-CONVENTIONS § 1.铁律一

### 2.4 TC-M0-004 平台字典筛选

- **步骤**：
  1. 在内容发布趋势图上选择"平台=抖音"
  2. 观察折线图
- **预期**：折线图仅显示抖音平台数据
- **来源**：AC-M0-001-4, GLOBAL-CONVENTIONS § 1.铁律二

### 2.5 TC-M0-005 手动刷新

- **步骤**：
  1. 进入首页
  2. 后台新增 1 个作者
  3. 等待 30s
  4. 点击"刷新"
- **预期**：作者数 +1
- **来源**：AC-M0-001-5

### 2.6 TC-M0-006 快捷入口权限

- **前置**：用"数据分析师"账号
- **步骤**：
  1. 进入首页
- **预期**：快捷入口仅显示"数据报表""指标管理"等；"账号管理""IP 组管理"不显示
- **来源**：AC-M0-001-6

### 2.7 TC-M0-007 关联属性校验（🔴 重点）

- **目的**：验证 GLOBAL-CONVENTIONS § 1.铁律一
- **步骤**：
  1. 调用 `/dashboard/home/metrics`，传 `ipGroupId=张三`（name 而非 id）
- **预期**：返回错误码 1500（关联实体不存在）
- **原因**：必须传 ID，不能传 name

### 2.8 TC-M0-008 字典 value 校验

- **步骤**：调用 `/dashboard/home/trend`，传 `platformType=INVALID`
- **预期**：返回错误码 1503（字典值不合法）

### 2.9 TC-M0-009 IP 组跨租户

- **步骤**：用租户 A 账号传 `ipGroupId=9999`（租户 B 的 IP 组）
- **预期**：返回错误码 1504

### 2.10 TC-M0-010 无权限

- **前置**：用游客账号
- **步骤**：访问首页
- **预期**：整页"无访问权限"
