## 1. 需求摘要

在 M2 **内容生产** 模块新增 **公推模板库**，用于管理 **公众号推文内容版式模板**：

- 手动新增 / 编辑 / 预览模板
- 从 **公众号文章链接** 或 **Word (.docx)** 导入版式
- 在 **新增内容**、**任务执行内容编辑** 中，当 `content_type=ARTICLE` 且类型匹配时可 **应用模板**
- **正文** 须支持 **富版式编辑 + 只读展示**（非纯文本 Textarea）
- **内容查看**、**内容审核** 须同样渲染富版式
