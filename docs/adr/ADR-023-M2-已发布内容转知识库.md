# ADR-023：M2 已发布内容转知识库

**状态**：已采纳（2026-06-14）

## 背景

已发布（`PUBLISHED`）的优秀内容需沉淀到知识库供团队复用，且每条内容仅允许转入一次。

## 决策

| 项 | 决策 |
|----|------|
| 前置条件 | `status=PUBLISHED` 且 `transferred_to_knowledge=0` |
| API | `POST /admin-api/oa/content/{id}/transfer-to-knowledge` |
| 知识库分类 | 固定 `CASE_LIB`（案例库） |
| 正文映射 | `body_format=LAYOUT` 且 `layout_html` 非空 → 用 `layout_html`；否则用 `body` |
| 标签 | 拼接 `content_type`、`document_type`、`platform_type(s)` |
| 公开性 | `is_public=1`（租户内可见） |
| 幂等 | 写入 `transferred_to_knowledge=1`、`knowledge_id`；重复请求返回 2040 |
| 权限 | `oa:content:transfer-knowledge` |

## 数据变更

`oa_production_content` 新增：

- `transferred_to_knowledge` TINYINT(1) DEFAULT 0
- `knowledge_id` BIGINT NULL → `oa_knowledge_base.id`

## 后果

- 已转内容列表展示「已转知识库」标签，隐藏操作按钮
- 与公推模板库、知识库「模板库」分类独立（见 ADR-019 / PRD FR-M2-005）
