# ADR-025：M4 公众号字段扩展与续费认证（V86）

**状态**：已采纳  
**日期**：2026-06-15  
**迁移**：`V86__m4_wechat_official_expand.sql`  
**关联**：FR-M4-005 · PRD-M4 §4.5

## 背景

公众号（`platform_type=WECHAT_OFFICIAL`）需独立维护认证/资质/管理员信息；续费认证需留痕（时间、续费人、金额）。

## 决策

### 1. 平台账号 `oa_account` 公众号扩展字段

仅当 `platform_type=WECHAT_OFFICIAL`（或服务号同类表单）时展示：

| 字段 | 控件 | 字典/实体 | 敏感 |
|------|------|----------|------|
| `trademark_name` | `<Input />` | - | - |
| `email` | `<Input />` | - | - |
| `password_encrypted` | `<Input type=password />` | AES-256 | ✅ |
| `qualification_type` | `<DictSelect dict-type="dict_qualification_type" />` | 企业/个人 | - |
| `usage_status` | `<DictSelect dict-type="dict_wechat_usage_status" />` | 注册/认证/续费 | - |
| `original_account_name` | `<Input />` | - | - |
| `cert_expiry_time` | `<DateTimePicker />` | - | - |
| `cert_count` | 只读/数字 | 认证次数累计 | - |
| `linked_video_account_id` | `<AccountSelect />` | 同租户视频号 `oa_account` | - |
| `video_account_registered_at` | `<DateTimePicker />` | - | - |
| `admin_name` | `<Input />` | - | - |
| `admin_user_id` | `<UserSelect />` | `sys_user` | - |
| `admin_id_card_encrypted` | `<Input />` | AES-256 | ✅ |

`dict_account_type` 新增 `SUBSCRIPTION_ACCOUNT`（订阅号）。

### 2. 条件表单（企业 / 个人）

| `qualification_type` | 展示字段 |
|---------------------|----------|
| `ENTERPRISE` | `company_id`（`<CompanySelect />` 必填） |
| `PERSONAL` | 实名人相关字段（沿用平台账号实名人选择器）；**隐藏**企业主体 |

切换资质类型时前端清空不适用的 `company_id` / 个人专属字段；后端 update 时按类型丢弃互斥字段。

### 3. 续费认证子表 `oa_wechat_official_cert_renewal`

| 字段 | 控件 | 说明 |
|------|------|------|
| `account_id` | 隐式 | 关联公众号 |
| `renewal_time` | `<DateTimePicker />` | 必填 |
| `renewer_user_id` | `<UserSelect />` | 续费人，可选 |
| `renewal_amount` | `<InputNumber />` | 默认 300.00 |

**API**（租户隔离）：

- `GET /admin-api/oa/account/wechat-cert-renewal/list?accountId=`
- `POST /admin-api/oa/account/wechat-cert-renewal/create`
- `DELETE /admin-api/oa/account/wechat-cert-renewal/{id}`

UI：`PlatformAccountDetail.vue` 公众号详情 Tab 内「续费认证记录」表格 + 新增弹窗。

### 4. 与跨平台字段关系

通用平台账号字段（实名人/手机/手机卡/IP 组等）不变；公众号扩展字段与通用字段并存，详情页分区展示。
