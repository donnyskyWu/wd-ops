# ADR-021：富文本编辑器与版式双向同步（FR-134/135）

**状态**：已采纳（§5 增补 2026-06-15）  
**日期**：2026-06-14  
**关联**：FR-134、FR-135、**FR-143**、**FR-147**、ADR-019、ADR-020

## 背景

公推模板编辑需支持富文本 WYSIWYG 与结构槽位编辑双模式；内容创作（ARTICLE）正文需富文本编辑，套用模板后即时可见样式效果。

## 决策

### 1. 前端富文本栈

- 采用 **TipTap**（`@tiptap/vue-3` + `starter-kit` + `underline` / `text-align` / `placeholder`）
- 封装 `RichTextEditor.vue`（通用）与 `LayoutRichEditor.vue`（模板编辑，含说明文案）

### 2. 双向同步（`layoutSync.ts`）

客户端镜像后端 `LayoutSchemaHelper` / `LayoutJsonHelper`：

| 方向 | 函数 | 说明 |
|------|------|------|
| schema → HTML | `schemaToPreviewHtml` | 骨架占位符预览，供富文本模式 |
| HTML → schema | `htmlToLayoutSchema` | 解析 DOM → v1 blocks → 提取 v2 schema |
| HTML → layout_json | `parseHtmlToLayoutDocument` | 内容正文解析 |
| layout_json → HTML | `renderLayoutHtml` | 渲染展示用 HTML |
| HTML → 纯文本 | `extractPlainText` | 写入 `body` 供搜索/AI |

同步在模式切换与保存时触发；`syncing` 标志避免 watch 循环。

### 3. 内容存储语义（ARTICLE）

- `body_format = LAYOUT`
- **`layout_html`**：富文本展示 SSOT（编辑器主内容）
- **`layout_json`**：结构化块，供 merge / 右侧结构编辑
- **`body`**：由 `extractPlainText(layout_html)` 派生，供搜索与 AI 提示

后端 save DTO 不变；现有字段已支持 HTML。

### 4. 模板编辑页

`edit.vue` 使用 `el-tabs`：「富文本编辑」↔「结构编辑」，共享 `form.layoutSchema`；保存前若处于富文本模式则先 `htmlToLayoutSchema`。

## 限制（已知）

1. **HTML → schema 为 best-effort**：复杂 inline style、嵌套 section、自定义 class 可能无法完整还原为槽位/globalStyles
2. **重复段落槽**：多个 `<p>` 合并为一个 `repeat` paragraph slot（与 v2 import 一致）
3. **图片框**：空 src 图片解析为 frame slot；富文本内联图片样式可能丢失
4. **TipTap 输出**：仅保留 StarterKit 支持的标签子集；与微信原文 HTML 100% 互操作非目标

### 5. 微信 HTML 粘贴/导出（2026-06-15 · `wechatHtml.ts`）

| 函数 | 时机 | 行为 |
|------|------|------|
| `normalizeWechatPasteHtml` | 编辑器 `handlePaste` | 提取 `rich_media_content` / ProseMirror 正文；剥离 separator、trailingBreak、`leaf` span |
| `sanitizeWechatExportHtml` | 保存 / 导出 | 去 script、阻断 on* 事件、剥离鉴权 query；保留 `<section>` 供 `StyledSection` |
| `ensureImageWidthStyles` | `LayoutViewer` 渲染 | 将 `data-w` 同步为 inline `width`，查看态与编辑态图片尺寸一致 |

**工具栏**（`EditorToolbar.vue`）：对齐公众号常见能力 — 字号、颜色/背景、对齐、列表、引用、表格、图片上传（5MB）与宽度调整。

**编辑器 UX**（`ContentEditPanel`）：全屏编辑、版式侧栏默认收起、全屏时隐藏侧栏。

## 验证

- 模板：富文本改标题样式 → 切结构模式见 heading 槽更新
- 内容：ARTICLE 富文本录入 → 应用模板 → 编辑器显示 styled HTML；审核页 `LayoutViewer` 渲染
